Part 1 - Baseline tasks

This first set of tasks will measure how long it takes you to find the target sound without any visual assistance.
Instructions:
  1. Sounds are randomly placed.
  2. Find the target sound as quickly as possible!


Partie 1 - Tâches de base


Cette première série de tâches permettra de mesurer le temps nécessaire pour trouver le son ciblé sans aucune aide visuelle.
Instructions:
  1. La position des sons est aléatoire.
  2. Trouvez le son ciblé le plus rapidement possible!

Practice 1

Highlight all the grey circles by passing your mouse cursor over them.
While doing this, note that circles close together produce similar sounds.
Retry this task to improve your sense of where certain sounds are located.

Pratique 1

Changez la couleur de tous les cercles gris en passant le curseur de votre souris dessus.
Ce faisant, notez que les cercles rapprochées produisent des sons similaires.
Réessayez cette tâche pour améliorer votre perception de l'emplacement de certains sons.

Practice

In this practice you will find a target sound.
Instructions :
  1. Press the spacebar to listen to the target sound.
  2. Pass the mouse over the circles to find the target sound.
  3. Complete the task by clicking the correct circle.
  4. Take the opportunity to adjust the sound volume so that you are comfortable with the sound level.
The task will end when you click on the correct circle.

Once you complete it, you will be given the option to continue or retry it. Repeat the task until you are comfortable with how to interact with it.


Pratique

Dans cette pratique, vous entendrez un son à retrouver.
Instructions :
  1. Appuyez sur la barre d'espace pour entendre le son ciblé.
  2. Passez la souris sur les cercles pour retrouver ce son.
  3. Terminez la tâche en cliquant sur le cercle approprié .
  4. Prenez l’occasion d’ajuster le volume du son pour être à l’aise avec le niveau sonore.
La tâche se termine lorsque vous cliquez sur le cercle approprié.

Une fois que vous avez terminé, vous aurez la possibilité de poursuivre ou de réessayer. Répétez la tâche jusqu'à ce que vous soyez confortable avec cette procédure.



Remember:
  • Press the spacebar to listen to the target sound.
  • The task will end when you click on the correct circle.


Rappel :
  • Appuyez sur la barre d'espace pour écouter le son ciblé.
  • La tâche se termine lorsque vous cliquez sur le bon cercle.

Instructions:
  1. Listen to circles that are close to one another. They share a sound trait.

  3. Use this information to locate the target sound.

  3. Utilisez ces informations pour localiser le son ciblé.
